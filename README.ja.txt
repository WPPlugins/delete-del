"Delete Del" は RSS/ATOM フィードから del 要素を、フィードおよび投稿・ページ内容から HTML コメントを削除します。

== 説明 ==

ins, del 要素は、投稿やページにおいて前の版と現在の版の違いを示すのに使われています。通常の XHTML+CSS 出力では、ins 要素は下線 (アンダーライン) で、del 要素は取消線で表示されます。
RSS 1.0 フィードでは、すべての XHTML マークアップが取り除かれます。ins や del マークアップも除去されます。RSS 1.0フィードの購読者は、ins や del マークアップを認識することができません。
ins マークアップは消されても安全ですが、del マークアップの除去は有害です。del 要素の中身のテキストも消去される必要があります。

このプラグインはそのような動作をします。つまり、RSS/ATOM フィードから del 要素の中身のテキストと共に削除します。


また、投稿やページの HTML コメントも、現在の WordPress HTML フィルターでは文法違反の原因となります。このために、このプラグインは、フィードおよび投稿・ページ内容から HTML コメントを削除します。

== 動作環境 ==

* WordPress 2.0 以降。
* PHP 4.2 以降。

== 設置方法 ==

1. プラグインアーカイブを展開し、「delete-del」フォルダーをサーバーのプラグインディレクトリー (wp-content/plugins/) に置きます。「delete-del.php」というPHPファイルのみをプラグインディレクトリーに置いても構いません。
2. プラグインを有効化します。
